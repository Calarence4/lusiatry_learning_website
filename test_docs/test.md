## 7. 后端 API 测试

### 7.1 文件树 API
# 获取文件树
curl http://localhost:3001/api/file-tree

# 确保草稿箱存在
curl http://localhost:3001/api/file-tree/draft-box

# 创建文件夹
curl -X POST http://localhost:3001/api/file-tree \
  -H "Content-Type: application/json" \
  -d '{"title":"测试文件夹","type":"folder"}'

# 创建笔记
curl -X POST http://localhost:3001/api/file-tree \
  -H "Content-Type: application/json" \
  -d '{"title":"测试笔记","type":"file","content":"# Hello"}'

### 7.2 API 响应检查
 GET /api/file-tree 返回 200
 GET /api/file-tree/draft-box 返回 200
 POST /api/file-tree 创建成功返回 201
 PUT /api/file-tree/:id 更新成功返回 200
 DELETE /api/file-tree/:id 删除成功返回 200
 删除系统草稿箱返回 403
## 8. 数据库检查

SELECT id, title, parent_id, LEFT(content, 50) as content_preview FROM file_tree WHERE type = 'file';
